# YKS 2026 Çalışma Platformu

GitHub Pages uyumlu statik frontend. İçerikler:

- 2018 TYT Türkçe, Sosyal Bilimler, Temel Matematik ve Fen Bilimleri için interaktif soru çözme ekranı
- Cevap seçmeden açılmayan çözüm panelleri ve yerel ilerleme takibi
- 2018-2025 TYT, AYT ve YDT PDF arşivi

## Yerelde çalıştırma

```bash
python3 -m http.server 8000
```

Ardından `http://localhost:8000` adresini açın. PDF arşivi için `http://localhost:8000/pdfler.html` sayfasına gidin.
